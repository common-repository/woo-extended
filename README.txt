== Description ==

The plugin "Woo Extended for Woocommerce" add several options useful and needed in Woocommerce

= Features =
* All options is modulable in the settings of Woocommerce
* 1: Disable cart page and redirect directly to checkout
* 2: Remove SKU (back and front end)
* 3: Remove Categories of products (back and front end)
* 4: Remove Tags of products (back and front end)
* 5: Auto Check the checkbox terms and conditions in checkout page
* 6: Uncheck the checkbox for use the different address shipping
* 7: Remove the field Notes in order checkout
* 8: Only sold all product individually
* 9: Redirect the login page (or register) Wordpress to the page of my account Woocommerce

== Installation ==
From within WordPress' dashboard:

1. Go to Plugins -> Add New
2. Search for "Woo Extended for Woocommerce"
3. Click "Install"
4. Click "Activate"

Manually via FTP:

1. Upload the folder 'woo-extended' into the '/wp-content/plugins/' directory
2. Activate the plugin through the *Plugins* tab in WordPress
